Privacy Policy

Last Updated: April 7, 2025

Introduction

Welcome to Memory Lane's Privacy Policy. This document outlines how the Memory Lane browser extension collects, uses, and protects your information. We are committed to ensuring that your privacy is protected and that you have a transparent understanding of our data practices.

Memory Lane is designed with privacy in mind. Our primary goal is to enhance your browsing experience by helping you remember why you bookmarked pages, without compromising your privacy or security.

Required Permissions

Memory Lane requires certain browser permissions to function properly. Below is a detailed explanation of each permission and why it's necessary:

storage

This permission allows Memory Lane to save your bookmarks, notes, tags, and preferences locally on your device. Without this permission, the extension would not be able to remember your saved memories between browser sessions.

tabs

This permission allows Memory Lane to access information about your currently open tabs. It's necessary to detect when you're visiting a page that you've previously saved, so we can show you your notes and context.

bookmarks

This permission allows Memory Lane to interact with your browser's bookmark system. It's used to enhance your existing bookmarks with additional context and notes.

activeTab

This permission gives Memory Lane temporary access to the currently active tab when you click the extension icon. It's used to capture the URL, title, and optionally a snapshot of the current page when you save a memory.

scripting

This permission allows Memory Lane to inject content scripts into web pages. It's necessary for displaying the memory popup when you revisit a saved page.

host_permissions (<all_urls>)

This permission allows Memory Lane to work on any website you visit. It's necessary for detecting when you're visiting a previously saved page and for capturing page snapshots.

Information We Collect

Memory Lane collects and stores the following information:

  • URLs and page titles of websites you choose to save
  • Notes and tags that you add when saving a page
  • Timestamps of when you saved a page
  • Visual snapshots of pages (only if you enable this feature)
  • User preferences for extension settings

What we DO NOT collect:

  • Your browsing history beyond the pages you explicitly save
  • Personal information such as names, email addresses, or payment details
  • Information from form fields, passwords, or login credentials
  • Any data from incognito/private browsing sessions

How We Use Your Information

The information collected by Memory Lane is used exclusively for the following purposes:

  • To display your saved notes, tags, and context when you revisit a bookmarked page
  • To provide a searchable and filterable timeline of your saved memories
  • To customize the extension's behavior based on your preferences

We do not:

  • Sell or rent your data to third parties
  • Use your data for advertising or marketing purposes
  • Share your data with any external services unless explicitly requested by you (e.g., when using the Wayback Machine integration)

Data Storage & Security

Memory Lane stores all your data locally on your device using the browser's built-in storage APIs. By default, no data is transmitted to external servers.

Your data is stored in the following ways:

  • chrome.storage.local: Used to store your memories, notes, tags, and preferences
  • Local snapshots: Visual snapshots are stored as data URLs in local storage

We take reasonable precautions to protect your information, but please be aware that no method of electronic storage is 100% secure. We encourage you to regularly back up your data using the export feature.

Third-Party Services

Memory Lane may interact with the following third-party services if you enable the related features:

Internet Archive's Wayback Machine

If you enable the Wayback Machine integration, Memory Lane will send URLs of pages you save to the Internet Archive's Wayback Machine to create an archived snapshot. This is an optional feature that you can disable in the settings.

For more information about the Wayback Machine's privacy practices, please visit: Internet Archive Terms of Service

User Rights

As a user of Memory Lane, you have the following rights regarding your data:

  • Access: You can view all your saved memories through the extension's interface
  • Export: You can export your data at any time using the export feature
  • Modification: You can edit or update your saved notes and tags at any time
  • Deletion: You can delete individual memories or clear all data through the extension's interface
  • Uninstallation: Uninstalling the extension will remove all locally stored data

Changes to This Privacy Policy

We may update our Privacy Policy from time to time. We will notify you of any changes by:

  • Posting the new Privacy Policy on this page
  • Updating the "Last Updated" date at the top of this page
  • Displaying a notification within the extension when significant changes are made

We recommend checking this page periodically for any changes. Your continued use of Memory Lane after we post changes to the Privacy Policy will constitute your acknowledgment of the changes and your consent to abide by the updated terms.

Contact Information

If you have any questions or concerns about this Privacy Policy or Memory Lane's data practices, please contact us by:

  • Opening an issue on our GitHub repository
  • Contacting the developer directly through GitHub
