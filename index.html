About Memory Lane

Enhancing your browsing experience with context and memories

Have you ever bookmarked a page and later forgotten why you saved it? Memory Lane solves this problem by allowing you to add notes, tags, and even visual snapshots when you bookmark a page. When you revisit the page, Memory Lane shows you your original context, helping you remember exactly why you saved it in the first place.

Whether you're a researcher collecting sources, a developer saving code snippets, or just someone who wants to remember why a particular article was important, Memory Lane helps you maintain context across your browsing history.

Installation

Get started with Memory Lane in a few simple steps

1

Download the Extension

Clone or download the repository from GitHub: Memory Lane Repository

2

Open Chrome Extensions Page

Navigate to chrome://extensions/ in your Chrome browser

3

Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the extensions page

4

Load the Extension

Click "Load unpacked" and select the extension folder from the downloaded repository

5

Start Using Memory Lane

The extension is now installed and ready to use! Click the Memory Lane icon in your browser toolbar to get started

How to Use

Make the most of Memory Lane with these simple workflows

üìå Saving a Page

Click the Memory Lane icon in your browser toolbar, add notes and tags about why you're saving the page, then click "Save Memory". Optionally, you can take a snapshot of the current page state.

üîç Revisiting a Page

When you visit a previously saved page, a small popup will appear in the corner of your screen showing your notes, tags, and the date you saved it. You can minimize this popup or click to expand it for more details.

üìö Viewing All Memories

Click "View All Memories" in the popup to see a timeline of all your saved pages. You can filter by date, search for specific content, or browse by tags to find exactly what you're looking for.
